# Contributing to 橙皮书 (Orange Book)

感谢你对 Hermes Agent 橙皮书的关注！本文档说明如何为项目贡献。

## 反馈与建议

### 报告问题

如果你发现书中的错误或有改进建议：

1. **提交 Issue** - 在 [GitHub Issues](https://github.com/alchaincyf/hermes-agent-orange-book/issues) 中创建新问题
2. **使用模板** - 选择合适的模板（Bug 报告 / 内容建议）
3. **详细描述** - 包含章节号、页码、具体问题描述

### 内容纠错

书中内容可能存在以下问题：
- 技术描述不准确
- 代码示例无法运行
- 链接失效
- 错别字/排版问题

请在 Issue 中注明：
- 章节号
- 原文位置（页码/段落）
- 建议修改内容

## 贡献方式

### 小改动

- 错别字、链接修复等小改动，直接提交 PR 即可
- PR 描述中说明改动位置和原因

### 大改动

- 新增章节、重构内容等大改动，请先提交 Issue 讨论
- 确认方向后再开始工作

### 提交 PR

1. Fork 本仓库
2. 创建分支：`git checkout -b your-feature-name`
3. 提交改动：`git commit -m "描述你的改动"`
4. 推送分支：`git push origin your-feature-name`
5. 创建 Pull Request

## 内容规范

### 术语一致性

| 术语 | 统一译法 |
|------|----------|
| Agent | Agent（不翻译） |
| Skill | Skill（不翻译） |
| Harness | Harness（不翻译） |
| Tool | 工具 |
| Memory | 记忆 |
| Prompt | 提示词 |

### 代码示例

- 使用代码块包裹
- 注释使用中文
- 确保代码可运行

### 链接格式

- 官方文档：优先使用 hermes-agent.nousresearch.com
- GitHub：使用 https://github.com/NousResearch/hermes-agent
